## Descripción

Invitación digital interactiva diseñada para celebrar un primer cumpleaños con una estética inspirada en *Monsters, Inc.*.  
Incluye animaciones, diseño responsivo y una experiencia visual pensada tanto para dispositivos móviles como de escritorio.

La invitación inicia con una pantalla de acceso animada (puerta interactiva) que da paso al contenido principal del evento.

---

## Características

- Diseño completamente responsivo
- Animación de entrada con interacción
- Elementos visuales dinámicos (globos, luces, efectos)
- Integración con Google Maps
- Tipografía personalizada
- Experiencia optimizada para compartir por enlace

## Personalización

Para adaptar la invitación, edita directamente el archivo `index.html`:

- Nombre del homenajeado
- Fecha y hora del evento
- Nombre de los padres
- Lugar y dirección
- Enlace de Google Maps

También puedes modificar:

- Colores en variables CSS (`:root`)
- Tamaños y disposición de elementos
- Animaciones y efectos visuales

---

## Tecnologías utilizadas

- HTML5
- CSS3 (Flexbox, Grid, Animaciones)
- JavaScript (interacción básica)
- Google Fonts

---

## Publicación

El proyecto está desplegado utilizando GitHub Pages.

Para actualizar cambios:

1. Modifica el archivo `index.html`
2. Sube los cambios al repositorio
3. GitHub Pages actualizará automáticamente la versión publicada

---

## Notas

- Asegúrate de mantener los nombres de archivos exactamente iguales (respetando mayúsculas y minúsculas)
- Todas las imágenes deben estar en la raíz del proyecto
- Se recomienda probar en diferentes dispositivos antes de compartir
