## Descripción

Este proyecto es una **invitación digital interactiva** desarrollada para la celebración de un primer cumpleaños, inspirada en una estética visual colorida y dinámica.

La invitación está diseñada como una experiencia web, incorporando animaciones, transiciones y elementos visuales que generan una interacción más atractiva que una invitación tradicional.

---

## Tecnologías utilizadas

- HTML5  
- CSS3 (Flexbox, Grid, Animaciones)  
- JavaScript (interacción y eventos)  
- Google Fonts  

---

## Características

- Animación inicial con pantalla de entrada  
- Transición interactiva hacia la invitación  
- Diseño completamente responsive (móvil y escritorio)  
- Uso de efectos visuales (blur, sombras, gradientes)  
- Elementos decorativos animados  
- Estructura clara para personalización
